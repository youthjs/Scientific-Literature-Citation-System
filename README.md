# 智能文本分类识别系统
基于 TextCNN 算法实现的中文文本自动分类系统，支持对多领域中文文本进行快速识别与分类，可应用于新闻分类、舆情分析、内容审核等场景。

## 一、项目概述
本系统针对中文文本信息分类需求，采用 TextCNN 卷积神经网络模型作为核心分类算法，通过提取文本的局部语义特征实现端到端的文本分类任务。系统支持自定义数据集训练、模型保存与加载、单条/批量文本预测等功能，整体架构轻量、训练速度快、推理性能优异。

## 二、核心技术原理
### 2.1 TextCNN 算法流程
TextCNN 是适用于文本分类的轻量级卷积神经网络，核心执行流程如下：

1.  **词嵌入层**：将输入文本转换为固定维度的词向量矩阵，作为模型的输入特征。
2.  **卷积特征提取**：使用多种不同尺寸的卷积核对文本序列进行卷积运算，对应提取文本的 N-Gram 局部语义信息。不同尺寸的卷积核分别捕获 2-gram、3-gram、4-gram 等不同粒度的词组特征。
3.  **最大池化操作**：对每个卷积核的输出结果执行全局最大池化，提取每一路卷积中的最关键特征值，实现特征降维并突出核心语义信息。
4.  **特征拼接与分类**：将所有池化后的特征向量拼接为完整特征向量，输入全连接层进行特征组合与映射，最终输出文本对应各个类别的概率。
5.  **损失函数**：训练阶段采用交叉熵损失函数（Cross-Entropy Loss）优化模型参数，最小化预测类别与真实标签的误差。

### 2.2 N-Gram 说明
N-Gram 是一种基于统计的语言模型算法，指文本中连续出现的 N 个词语/字符组合。TextCNN 通过不同宽度的卷积核，天然实现了对不同 N 值词组特征的自动提取，无需人工构建特征工程。

## 五、使用说明
### 5.1 数据准备
将中文文本数据集放入 `train_pred/data/` 目录，格式为 `文本内容\t标签` 的制表符分隔文件，支持自定义类别标签。

### 5.2 模型训练
运行训练脚本，自动完成数据加载、分词、词向量构建、模型训练与保存：
```bash
python train_pred/main.py --mode train
```
训练完成后，模型文件将自动保存为 `TextCNN_model.h5`。

### 5.3 文本预测
加载已训练模型，对输入文本进行分类预测：
```bash
python train_pred/main.py --mode predict --input "待分类的中文文本内容"
```
系统将输出预测的文本类别及对应置信度。

## 六、模型特点
- **特征提取高效**：多尺寸卷积核并行提取不同粒度的语义特征，捕捉关键词组信息。
- **模型轻量快速**：网络结构简洁，训练与推理速度快，适合快速迭代与部署。
- **泛化能力强**：最大池化操作过滤冗余信息，模型对文本语序波动鲁棒性较好。
- **易于扩展**：支持自定义类别数、卷积核尺寸、词向量维度等超参数调整。

## 七、注意事项
1.  数据集文件体积较大，已通过 `.gitignore` 排除，请勿将大体积训练数据提交至代码仓库。
2.  若需更换数据集，需同步调整类别数量与标签映射关系。
3.  可根据文本长度与领域特性，调整卷积核尺寸、词向量维度等超参数优化分类效果。
